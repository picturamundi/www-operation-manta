# About this Handbook

See the Game Intro for an introduction to the Toyota Wars board game.

# Background

This Handbook was created to accompany the Toyota Wars board game designed by LtCol. Richard Davis, USAF, and published by Decision Games in Strategy and Tactics issue #144 (opens new window) (1991). The original edition of the game ships in a magazine format with basically everything you need to play, including the game map, rulebook, and game pieces.

The Handbook is meant to make Toyota Wars more accessible to players. It can be used to learn about Toyota Wars before purchasing the game, and, once game materials have been received from Strategy and Tactics, it can be used to navigate the rules and consult important charts and resources more efficiently than with the print magazine.

# Updates included in this Handbook

Rules

  • The rulebook now includes additional graphics, explanations, and introductory material for player comprehension.
  • While the rules have been somewhat reorganized for accessibility, the game is unchanged.
  • Best of all, because the rules in this Handbook are digitized, you can search them!

Dice

  • The rulebook includes a pair of animated six-sided dice. The dice are the only game component missing from the Strategy and Tactics magazine.

Map

  • A redesigned game map is also accessible on this site, and should be printed on US ANSI D paper (24"x32").
  • This Handbook is compatible with both the original 1991 map and the redesigned map.
