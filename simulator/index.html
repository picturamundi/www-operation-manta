# Opération Manta Simulator

# Getting Started

# Game Installation

  1. If you don't already have one, register for a Steam account (opens new window) and install the program.
  2. Log into Steam. In the top menu, select Store. Search for "Table Top Simulator." Purchase and install the simulator ($20). This will provide our game environment.
  3. There are two ways to install the Operation Manta scenario.
    • If using the Steam app, select Community > Workshop. Search for "Operation Manta." A workshop item by user HassanDjamous should show up. This is the game mod that will allow us to play the Toyota Wars board game in the simulator.
    • Alternatively, log into Steam in a web browser and access the workshop mod via this link (opens new window).
  4. Select the workshop item and click the green Subscribe button.

# Play

  1. Launch Table Top Simulator from your desktop applications folder.
  2. To host a game, select Create Game on the simulator launch screen and click Multiplayer. Name the game and launch the server.
  3. Once the game environment launches, we need to load Toyota Wars. Select Games in the top menu. A game menu will pop up and display workshop subscriptions. Select Opération Manta.
  4. Once Opération Manta loads, we can invite the second player. Click the + under your username on the top-right. Select a friend to invite. Alternatively, friends can brows and search game servers from the launch screen.
  5. Tap the / or ? key to access basic commands in Table Top Simulator, or visit the TTS tutorial series (opens new window) on youtube.
  6. Use the game rules on this website to navigate gameplay. If you've never played before, begin with the game intro.
  7. If you're playing the game on and off with a friend, it can be useful to use the large table tile to indicate who's turn it is.

# Pause

  1. If you want to quit the game and come back to it later, one of the players needs to save a local instance that they can host at a later time. In the top menu, select Games > Save and Load > Save game.
  2. When picking back up the game, follow the same steps as above, only instead of loading the workshop mod, the player hosting the game will load the local save, which will also be listed in the game menu along with subscriptions.
